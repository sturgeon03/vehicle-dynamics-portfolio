# Vehicle Dynamics and Control Portfolio

Public portfolio site for `Vehicle Dynamics and Control, 2nd Edition` study notes, topic maps, and simulation-oriented explanations.

## Repository Role

- This repository is public-facing and contains only portfolio-safe content.
- Working translations, raw PDF assets, and detailed private study drafts stay in a separate private repository.

## Local Development

```powershell
python -m venv .venv
.\.venv\Scripts\python -m pip install -r requirements.txt
.\.venv\Scripts\mkdocs serve
```
